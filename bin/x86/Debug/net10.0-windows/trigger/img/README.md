# Thư mục ảnh mẫu quét màn hình

Đặt ảnh mẫu mặc định tại đây:

```text
trigger\img\template.png
```

Khi build/publish, các file PNG trong thư mục này sẽ được copy sang thư mục chạy của chương trình.

Ảnh mẫu nên nhỏ, rõ, ổn định và đúng vùng hình ảnh cần nhận diện trên màn hình. Không dùng ảnh có số, thời gian, animation hoặc nội dung thay đổi liên tục.
